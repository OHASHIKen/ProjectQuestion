# Project Question

## 趣旨説明
このリポジトリは、質問の仕方やそのトレーニング、応用を、できる限り多くの人が文字により理解し、身につけることを目的としたプロジェクトのメインページです。

## 構成
このプロジェクトは、大きく分けて以下の4つにより構成されます。
* 「質問」のここでの定義と、その方法の文字説明
* 質問の具体例
* 質問のやり方を身につける方法
* 質問の効果、意義、応用
それぞれの項目には、対応する評価レポートがあり、それぞれの説明の質と一般性、修正すべき点、問題点の指摘が行われます。

## 作成手順
コンテンツの作成手順は以下の通りです。
* コンテンツ作成のリクエストを送る。
* 運営者が、コンテンツ作成の領域をDraft内に作る
* 作成者は、自分のコンテンツを否定できる可能性をあげて検討し、作成したコンテンツの有効範囲を見積もる。
* これらの内容について、コメントを募集し、修正する。
* 管理者が、コンテンツの内容及び有効範囲が適切と認めた段階で、これらの内容を一つのマークダウンファイルにまとめ、Content内に置く。
* Draft内の議論が進展した場合、その内容を反映するためのリクエストを送り、Content内のファイルを修正する。
